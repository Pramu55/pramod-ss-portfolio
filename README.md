# Pramod S S - Portfolio

Personal portfolio website showcasing my work as a DevOps Engineer and Full-Stack Developer.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## Features

- Modern, responsive design with glassmorphism effects
- Smooth gradient backgrounds and hover animations
- Sections: Hero, Skills, Experience, Projects, Education
- Optimized for performance and SEO

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
